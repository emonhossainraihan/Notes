## Proxy Use Cases 

```
client > proxy > server
```

- Logging
- Anonymity
- Geofencing 
- Caching
- Block sites
- Enable Polyglot 

## Reverse Proxy Use Cases 

```
                                /-> server-1
client > server > reverse-proxy 
                                \-> server-2
```

- Caching 
- Load Balancing 
- Ingress
- Canary Deployment
- Microservices
